\subsection*{Problème}

J'ai mon père à faire les courses. Je récupère le ticket de caisse avec les calculs.
\begin{itemize}
  \item $3 \times 2.65 = 7.95$
  \item $2 \times 3.42 = 6.84$
  \item $1.65 \times 2.4 = 3.96$
  \item $6.84 + 3.96 + 1.17 = 19.92$
  \item $20 - 19.92 = 0.08$
\end{itemize}
